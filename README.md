# 🎬 Telegram Movie Bot

## 📋 Bot haqida
Ushbu Telegram bot foydalanuvchilarga kinolarni qidirish, ko'rish va yuklab olish imkoniyatini beradi. Bot majburiy a'zolik, kino reytingi, qidirish va boshqa ko'plab funksiyalarga ega.

## 👤 Foydalanuvchi imkoniyatlari

### 🚀 Asosiy funksiyalar
- 🎬 **Kino kodlari** - Barcha kinolar kodlarini ko'rish
- 🔍 **Nomi bilan qidirish** - Kino nomini kiritib qidirish
- 🎯 **Bugunlik tavsiya** - Tasodifiy kino tavsiyasi
- 🏆 **Top reyting** - Eng yuqori reytingdagi kinolar ro'yxati
- ⭐ **Kinolarni baholash** - Kinolarga 1-5 yulduzli reyting berish
- 📤 **Do'stlarga ulashish** - Kinolarni do'stlarga ulashish

### 🔎 Kino qidirish
**Kod orqali qidirish:**
1. Botga kino kodini yuboring (masalan: 123)
2. Bot sizga shu kodli kinoni yuboradi

**Nom orqali qidirish:**
1. "🔍 Nomi bilan qidirish" tugmasini bosing
2. Kino nomini kiriting
3. Natijalardan kerakli kinoni tanlang

### 🎲 Tasodifiy kino
- `/rand` buyrug'ini yuboring yoki "🎯 Bugunlik tavsiya" tugmasini bosing
- Bot tasodifiy kinoni taqdim etadi

### ⭐ Kinolarni baholash
1. Kinoni ko'rayotganingizda "⭐ Kinoni baholash" tugmasini bosing
2. 1 dan 5 gacha yulduzli reyting bering
3. Xohishingizga ko'ra izoh qoldiring

### 📱 Boshqa buyruqlar
- `/start` - Botni ishga tushirish
- `/help` - Yordam olish
- `/rand` - Tasodifiy kino olish
